# Parth Mandore — Personal Portfolio Website

A premium, modern, and highly interactive developer portfolio website designed with clean aesthetics, responsive layout, and smooth animations. 

✨ **Live Demo:** [Deploy with Vercel or GitHub Pages]

---

## 🎨 Features & Visual Highlights

* **3D Showcase Card:** An Apple-style profile card centerpiece that tilts dynamically in Z-space on mouse hover, featuring a pop-out cutout portrait that blends seamlessly with a blurred backdrop.
* **Dual Modern Themes:**
  * **Dark Theme:** A deep purple/indigo space galaxy aesthetic with subtle rotating border glows.
  * **Light Theme:** A clean, LinkedIn-inspired professional white/blue layout with high-contrast typography.
* **Interactive Typewriter Header:** A synchronous code-editor block typewriter cursor that types `// Computer Engineering Student` and vanishes cleanly after a single blink upon completion.
* **Scroll-Triggered Counter Stats:** Numbers for DSA problems solved, CGPA, and projects count up rapidly from zero only when they enter the viewport.
* **Dynamic Mobile Navigation:** A responsive, compact 2x2 grid navigation menu with glassmorphic buttons that slide in smoothly and push page content down without overlapping logo/controls.
* **Recruiter-Friendly Contact Form:** A fully functional, asynchronous contact form powered by Web3Forms for direct inbox delivery.

---

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** CSS3 (Vanilla CSS variables, flexbox, grid, keyframe animations, glassmorphism)
* **Logic:** Vanilla JavaScript (ES6+, IntersectionObserver API for viewport scroll triggers)
* **Icons:** [Devicon](https://devicon.dev/) (SVG logo integration)
* **Fonts:** [Google Fonts](https://fonts.google.com/) (Outfit, Inter, Space Mono, Syne)

## 🚀 How to Run Locally

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/parthmandore/portfolio.git
   cd portfolio
   ```

2. **Open the project:**
   Simply double-click `index.html` to open it in any modern browser, or run a local dev server (like Live Server in VS Code) for live reloading.

## 🌐 Deployment Instructions

### Deploying to Vercel (Recommended)
1. Push your code to a GitHub repository.
2. Sign in to [Vercel](https://vercel.com/) and click **Add New Project**.
3. Import your repository.
4. Click **Deploy**. Vercel will automatically detect the static project and host it within seconds.

### Deploying to GitHub Pages
1. Go to your repository settings on GitHub.
2. Navigate to **Pages** in the sidebar.
3. Select the `main` or `master` branch under the **Build and deployment** source dropdown.
4. Click **Save**. Your site will be published at `https://<username>.github.io/<repository-name>`.
